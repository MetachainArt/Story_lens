# Story Lens 화면 설계

> Story Lens - 장애인을 위한 사진 편집 웹앱의 화면 명세

이 문서는 /screen-spec의 입력 자료입니다.

## 화면 목록

### 1. 로그인 화면 (/login)
- **진입**: 앱 시작 (미인증 상태)
- **요소**:
  - 앱 로고/이름
  - 아이디 입력 필드
  - 비밀번호 입력 필드
  - 로그인 버튼 (크고 명확)
- **액션**: 로그인
- **이동**: 로그인 성공 → 홈 화면
- **인증**: 선생님이 만들어준 계정 사용
- **에러**: 로그인 실패 시 간단한 안내 메시지

### 2. 홈 화면 (/)
- **진입**: 로그인 성공, 저장 완료 후 '홈으로'
- **요소**:
  - '사진 찍기' 큰 버튼 (메인 CTA)
  - '내 사진 보기' 버튼
  - 간단한 인사말 또는 앱 이름
- **이동**:
  - '사진 찍기' → 카메라 화면
  - '내 사진' → 사진 목록 화면
- **인증**: 필요 (로그인 상태)

### 3. 카메라 화면 (/camera)
- **진입**: 홈 → '사진 찍기' 버튼
- **요소**:
  - 카메라 뷰파인더 (전체 화면)
  - 촬영 버튼 (크고 둥근)
  - '찍기 끝' 버튼
  - 촬영 카운터 (찍은 사진 수)
- **액션**: 사진 촬영, 촬영 종료
- **이동**: '찍기 끝' → 사진 선택 화면
- **권한**: 카메라 권한 필요

### 4. 사진 선택 화면 (/select)
- **진입**: 카메라 → '찍기 끝'
- **요소**:
  - 사진 크게 보기 (스와이프로 전환)
  - 사진 번호 표시 (1/5 등)
  - '이 사진 편집하기' 버튼
  - '다시 찍기' 버튼
- **액션**: 스와이프로 사진 탐색, 편집할 사진 선택
- **이동**:
  - '이 사진 편집' → 편집 화면
  - '다시 찍기' → 카메라 화면

### 5. 편집 화면 (/edit/:photoId)
- **진입**: 사진 선택 → '이 사진 편집', 사진 목록 → 사진 선택
- **요소**:
  - 사진 (풀스크린 미리보기)
  - 하단 탭 3개:
    - **필터 탭**: 느낌 이름 필터 카드들 (따뜻한, 시원한, 흑백, 빈티지 등)
    - **조절 탭**: 슬라이더 5개 (밝기, 채도, 대비, 온도, 선명도)
    - **자르기 탭**: 자르기 영역 선택, 회전 버튼, 뒤집기 버튼
  - '저장' 버튼 (상단 또는 하단)
  - '취소/뒤로' 버튼
- **액션**: 필터 적용, 슬라이더 조절, 자르기/회전/뒤집기, 저장
- **이동**: '저장' → 저장 완료 화면

### 6. 저장 완료 화면 (/saved)
- **진입**: 편집 → '저장'
- **요소**:
  - '저장 완료!' 메시지
  - 편집된 사진 미리보기
  - '홈으로' 버튼
  - '다시 편집' 버튼
- **이동**:
  - '홈으로' → 홈 화면
  - '다시 편집' → 편집 화면

### 7. 사진 목록 화면 (/gallery)
- **진입**: 홈 → '내 사진'
- **요소**:
  - 격자 모양 갤러리 (2~3열)
  - 각 사진 썸네일
  - '뒤로가기' 버튼 (→ 홈)
- **액션**: 사진 터치하면 편집 화면으로 이동
- **이동**: 사진 선택 → 편집 화면

## 화면 간 연결 다이어그램

```
로그인 ──→ 홈
              ├──→ 카메라 ──→ 사진 선택 ──→ 편집 ──→ 저장 완료
              │                                ↑        │
              │                                │        ├──→ 홈
              │                                │        └──→ 편집 (다시)
              └──→  사진 목록 ─────────────────┘
```
